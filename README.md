# react-native-ui

## Requirement
https://facebook.github.io/react-native/docs/getting-started

https://expo.io/learn

## Install:
Clone the Repository
```bash
git clone https://github.com/prakhargvp/react-native-ui
```

Install the NPM Packages
```bash
npm install
```
To Start Expo
```bash
expo start
```
